## 🎨 Design Features

### Color Palette (Pink Boho + Earthy)
- **Terracotta** (#C97755): Primary action, energy, headlines
- **Sage** (#9CAF88): Trust, calm, secondary actions
- **Dusty Rose** (#D4A5A5): Warmth, accent
- **Deep Charcoal** (#3A3A3A): Primary text, contrast
- **Off-white** (#F5F3F0): Default backgrounds
- **Cream** (#FBF9F6): Card backgrounds, warmth
- **Soft Taupe** (#E8DDD6): Section separation

### Typography
- **Headlines**: Playfair Display (serif) - elegant, boutique
- **Body**: Inter (sans-serif) - modern, readable
- Hierarchy: H1 (3.5rem) → H2 (2.8rem) → H3 (1.5rem) → Body (1rem)

### Visual Aesthetic
- Fine-line icons and botanical graphics
- Generous whitespace and breathing room
- Soft shadows (not harsh)
- Rounded corners (12px) for soft feel
- Left-border accents on cards (4px)
- Smooth transitions and hover states (0.3s ease)
- Gradient backgrounds for CTAs
